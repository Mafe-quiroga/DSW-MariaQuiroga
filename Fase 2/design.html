Sitio web para Flex Box, Listas y enlaces

El proposito de este sitio web es favorecer el aprendizaje con respecto a los temas de Flex Box, Listas y enlaces

Flex Box

El Flex Box permite acomodar los elementos de una página según cambien las dimensiones y orientación de la página.Por tanto,
nos permite hacer diseño adaptivo para que nuestras páginas web se visualicen correctamente en cuanquier dispositivo. Este
estándar es bastante reciente y todavía no está soportado por todos los navegadores, aunque lo estara muy pronto. Las ventajas
son un código mucho más legible y simple, donde podemos, por ejemplo,cambiar el orden de los elementos independientemente del
orden en el codigo HTML.
